# 头条后台管理系统

一个基于 Vue3 + element plus 的后台管理项目

接口文档：https://www.yuque.com/lipengzhou/toutiao-publish-vue

## 安装步骤

克隆到本地

```
git clone git@github.com:Twitzz/toutiao-admin.git 
```

进入项目目录

```
cd toutiao-admin
```

安装项目依赖

```
npm install
```

开启本地服务

```
npm run serve
```
